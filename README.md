# Curandera Netlify Starter

This is a static landing page for Curandera, designed to feel earthy, ritualistic, refined, and modern.

## Files
- `index.html`
- `styles.css`
- `netlify.toml`

## Deploy to Netlify
1. Create a new GitHub repo and upload these files.
2. In Netlify, choose **Add new project** and connect the repo.
3. Build command: leave blank
4. Publish directory: `.`
5. Deploy.

This site also includes a Netlify form:
- form name: `waitlist`

## What this page includes
- Brand-forward hero
- Ritual/philosophy sections
- First collection section
- Waitlist form
- Netlify-ready configuration
